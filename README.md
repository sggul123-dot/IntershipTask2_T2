# Memory Puzzle Game using Python

## Project Title

Memory Puzzle Game

## Description

The Memory Puzzle Game is a simple Python-based game where players try to match pairs of cards. The cards are placed face down, and the player selects two cards at a time. If the cards match, they remain revealed; otherwise, they are hidden again. The goal of the game is to match all pairs with the fewest attempts.

This project helps in understanding basic Python programming concepts such as loops, lists, conditions, and user input handling.

## Objectives

* Create a simple interactive puzzle game
* Practice Python programming logic
* Use lists and randomization
* Improve problem-solving skills

## Technologies Used

* Python 3
* Built-in Python libraries (random)

## Required Libraries

No external libraries are required for this project.

## How the Program Works

1. A list of cards is created with pairs of values.
2. The cards are shuffled randomly.
3. All cards are displayed as hidden symbols (*).
4. The player selects two card indexes.
5. If the cards match, they remain visible.
6. If the cards do not match, they remain hidden.
7. The game continues until all pairs are matched.

## How to Run the Project

1. Install Python on your computer.
2. Save the program file as `memory_game.py`.
3. Open Command Prompt or Terminal.
4. Navigate to the folder where the file is saved.
5. Run the program using:

py memory_game.py

## Output

* Displays hidden cards using `*`
* Player selects card positions using indexes
* The program checks whether the cards match
* Game ends when all pairs are found
